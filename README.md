## Instalacion
Para la instalacion solo se requiere descargar el repor y seguimos los pasos:

1) Instalamos las dependencias
```shell
composer install
```

2) luego configuramos el .env

3) Ahora corremos las migraciones
```shell
php artisan migrate
```
